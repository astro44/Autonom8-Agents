## Self-Validation Loop (CRITICAL)

**IMPORTANT**: After implementing Terraform, you MUST validate using Terratest before declaring complete.

### Validation Workflow

```
┌─────────────────────────────────────────────────────────────────┐
│                     IMPLEMENT + VALIDATE                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. Write Terraform Code                                         │
│     └── Create/modify *.tf files                                │
│                                                                  │
│  2. Create Terratest                                             │
│     └── tests/{module}_test.go                                  │
│         - terraform.InitAndApply                                 │
│         - Assert outputs and resources                           │
│                                                                  │
│  3. Run Terratest                                                │
│     └── go test ./tests -v                                      │
│                                                                  │
│  4. If Tests FAIL:                                               │
│     └── Read error output                                        │
│     └── Fix module or test                                      │
│     └── Re-run tests (go to step 3)                              │
│                                                                  │
│  5. If Tests PASS:                                               │
│     └── Run terraform validate                                  │
│     └── Declare implementation complete                          │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### Test Commands

```bash
# Run all tests
go test ./... -v

# Run specific test file
go test ./tests -run Test{{MODULE_NAME}} -v

# Run terraform fmt and validate
terraform fmt -recursive
terraform validate
```

### Expected Output

```

--- PASS: TestNetworkModule (45.12s)
PASS
ok      tests   45.139s
```

```
This object has no argument, nested block, or exported attribute named "id".
```

## Terratest Module Test Template

```go
// tests/{{module_name}}_test.go
// Terratest template for {{MODULE_NAME}}
// Generated by Autonom8 Sprint Execution

package tests

import (
	"testing"

	"github.com/gruntwork-io/terratest/modules/terraform"
	"github.com/stretchr/testify/assert"
)

func Test{{MODULE_NAME}}(t *testing.T) {
	t.Parallel()

	terraformOptions := &terraform.Options{
		TerraformDir: "{{MODULE_DIR}}",
		Vars: map[string]interface{}{
{{#TF_VARS}}
			"{{VAR_NAME}}": "{{VAR_VALUE}}",
{{/TF_VARS}}
		},
	}

	defer terraform.Destroy(t, terraformOptions)
	terraform.InitAndApply(t, terraformOptions)

	output := terraform.Output(t, terraformOptions, "{{OUTPUT_NAME}}")
	assert.NotEmpty(t, output)
}
```

## Terratest Plan Test Template

```go
// tests/{{module_name}}_plan_test.go
// Terratest plan template for {{MODULE_NAME}}
// Generated by Autonom8 Sprint Execution

package tests

import (
	"testing"

	"github.com/gruntwork-io/terratest/modules/terraform"
	"github.com/stretchr/testify/assert"
)

func Test{{MODULE_NAME}}Plan(t *testing.T) {
	t.Parallel()

	terraformOptions := &terraform.Options{
		TerraformDir: "{{MODULE_DIR}}",
		Vars: map[string]interface{}{
{{#TF_VARS}}
			"{{VAR_NAME}}": "{{VAR_VALUE}}",
{{/TF_VARS}}
		},
		NoColor: true,
	}

	plan := terraform.InitAndPlanAndShowWithStruct(t, terraformOptions)
	assert.True(t, len(plan.ResourceChanges) > 0)
}
```
